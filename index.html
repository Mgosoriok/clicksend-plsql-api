Oracle PL/SQL API for Clicksend

Send transactional messages from your Oracle database to anyone with a phone.

View on GitHub Download Latest Release WIKI

The API provides methods to use your Clicksend account to send messages via:

  • SMS (Short Message Service)

  • MMS (Multimedia Message Service)

  • Text to Voice

For installation instructions, detailed API references and example code, refer to the WIKI.
